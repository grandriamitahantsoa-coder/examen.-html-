inana HTML

Ity dia tetikasa momba ny fanadinana HTML nataoko teto amin'ny GitHub.

## Inona no hita ato?
* Rakitra `index.html` misy ny vokatry ny fanadinana.
* Sary sy fitaovana hafa ampiasaina amin'ny tranonkala.

## Ahoana no hijerena azy?
Azonao sokafana amin'ny mpitety tranonkala (browser) fotsiny ny rakitra `index.html` raha vao avy alaina (download) ity tetikasa ity.
